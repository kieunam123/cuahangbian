== Description ==

This popup plugin comes with easy to use layouts, You can simply select and add your original content using live editor. Plugin contain layouts for sale, Discount offers, Deals, shop ad etc. You can popup at your desired page, post, Between post and in the widget areas as a banner ad.

**Key Features:-**

* Live popup editor.
* Pre made templates.
* Option to select desired page / post to display popup.
* Option to display popup as a banner ad in the widget area.
* Option to display popup between page / post using shortcode.
* Unlimited color option for popup content.
* Watch template demo live before editing.   
* 24x7 support available for free product too [http://www.themehunk.com/support](http://www.themehunk.com/support). 


== Installation ==

1. Upload the `business-popup.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Get started from Dashboard > Business Popup
